 Legacy Engineering Projects (Electronic & Embedded Systems)

این مخزن شامل مجموعه‌ای از پروژه‌های قدیمی و آرشیوی من در زمینه طراحی الکترونیک و برنامه‌نویسی میکروکنترلرها است. هدف از این مخزن، نگهداری تاریخچه فنی و مستندسازی تجربیات گذشته است.

## 🛠 ابزارهای استفاده شده (Technologies & Tools)
در این پروژه‌ها از ابزارهای زیر استفاده شده است:
- **Hardware Design:** Altium Designer (PCB & Schematic)
- **Firmware Development:** Keil uVision, STM32CubeIDE, CodeVisionAVR
- **Simulation:** Proteus ISIS/ARES
- **Languages:** C, C++, Python (Scripting)

## 📁 ساختار مخزن (Project Structure)
هر پوشه شامل یک پروژه مجزا است که معمولاً شامل موارد زیر می‌باشد:
- فایل‌های سورس کد (Source Code)
- شماتیک و فایل‌های PCB (در صورت وجود)
- فایل‌های شبیه‌سازی (Simulation files)

> **Note:** فایل‌های خروجی حجیم مثل `History/` و فایل‌های کامپایل شده (`.o`, `.hex`, `.bin`) برای بهینه‌سازی مخزن طبق فایل `.gitignore` فیلتر شده‌اند.

## ⚠️ وضعیت پروژه‌ها
این پروژه‌ها جنبه آرشیوی دارند و ممکن است در حال حاضر توسعه فعالی روی آن‌ها انجام نشود. با این حال، استفاده از ایده‌ها یا کدهای موجود برای اهداف آموزشی بلامانع است.
